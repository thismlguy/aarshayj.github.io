  • OPEN SOURCE

  • easyML - Code Less, Do More!!

    I'm developing a python package - easyML, designed to streamline the process of analyzing datasets using predictive models. It covers crutial aspects of data analysis process starting from data preprocessing, to feature engineering and finally predictive modeling.

    They key advantage of this package is the data block feature using which you can create a block of your data at the start of analysis. The module functions seemlessly integrate with all your data. The package is particularly useful for begineers and intermediate level python data science enthusiasts, who are looking to get the job done without worrying too much about the code. The package is an alpha-development stage and I'm adding new features regularly.
